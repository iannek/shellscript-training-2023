# 課題1. 【引数とコマンド実行の取り扱い】ディレクトリ使用率を表示するスクリプトを書く

## 実施すること

- 引数にディレクトリ名を渡して該当箇所の使用率を表示するスクリプトを実装してみましょう

## 実行例

```bash
./スクリプト名.sh /home
```

## 出力例

```
Use% 45%
```

## 動作確認(テスト)観点

- 存在するディレクトリを指定して正しく使用率が表示されること
- 存在しないディレクトリを指定した場合、エラー結果が表示されること
- 引数に指定した個数が誤っていた場合、エラー結果が表示されること
